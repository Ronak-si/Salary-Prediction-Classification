# 💼 Salary Prediction Classification

A Machine Learning project that predicts whether a person's salary belongs to a particular salary category based on demographic and employment-related features. The project includes data preprocessing, exploratory data analysis (EDA), feature engineering, model training, evaluation, and comparison of multiple classification algorithms.

---

## 📌 Project Overview

The objective of this project is to classify individuals into salary categories using machine learning techniques. The dataset is cleaned, analyzed, and transformed before training different classification models to identify the best-performing model.

---

## 🎯 Objectives

- Perform data cleaning and preprocessing.
- Conduct Exploratory Data Analysis (EDA).
- Encode categorical variables.
- Train multiple classification models.
- Compare model performance using evaluation metrics.
- Select the best model for salary prediction.

---

## 📂 Dataset

The dataset contains demographic and employment-related information such as:

- Age
- Workclass
- Education
- Marital Status
- Occupation
- Relationship
- Race
- Gender
- Capital Gain
- Capital Loss
- Hours per Week
- Native Country
- Salary (Target Variable)
